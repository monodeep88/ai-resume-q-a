# Resume Q&A Assistant

Type: **RAG**

Resume Q&A is a complete, runnable Python starter project generated by AI Project Generator. It is intentionally dependency-light so it can run anywhere, but it includes real application code rather than only folders.

## Included

- Agent behavior tailored for `RAG`
- Local knowledge base in `data/knowledge_base.json`
- Retrieval, planning, multi-agent, or tool-routing logic depending on the project type
- Command line interface
- Automated pytest coverage
- Clear extension points for adding a real LLM, vector database, web UI, or API

## Run locally

```bash
pip install -r requirements.txt
pytest
python -m resume_q_a.cli "help me solve a real user problem"
```

## How to customize

- Add domain documents to `resume_q_a/data/knowledge_base.json`
- Extend `resume_q_a/agents.py` with API calls or richer workflows
- Add more tools in `resume_q_a/tools.py`
- Add tests for every new behavior in `tests/`
